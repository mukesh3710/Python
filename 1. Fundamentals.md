# ðŸ”° **6. Functions**

### Simple:


### Real-world (parse patch group):

```python
def parse_group(hostname, group, slot):
    group = group.lower().replace(" ", "_").replace("-", "_")
    slot = slot.lower().replace(" ", "_").replace("-", "_")

    return {
        "hostname": hostname,
        "group": group,
        "slot": slot
    }
```
